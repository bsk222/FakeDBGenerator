## ✨ À propos

**2xOneFakeDbGenerator** est le générateur de données factices **le plus puissant et le plus rapide** pour Windows/Linux/macOS.

- Jusqu'à **50 millions de lignes** en quelques secondes  
- Interface graphique moderne (CustomTkinter)  
- Templates réalistes français (CAF, Impôts, ANTS, SFR, Banque, etc.)

- Emails avec vrais domaines (gmail, orange.fr, sfr.fr...)  
- 100 % offline • Aucun limite • Open Source

## 🚀 Tutoriel d'utilisation (2 minutes)

1. **Télécharge** le projet (ou clone le repo)
2. **Exécute** `start.bat` (Windows) ou `start.sh` (Linux/macOS)
3. L'interface s'ouvre automatiquement

### Comment générer ta base de données :

1. Choisis la **langue** (fr_FR par défaut)
2. Entre le **nombre de lignes** (ex: 5 000 000 ou 50 000 000)
3. Sélectionne le **format** (CSV, Excel, JSON, SQLite, Parquet)
4. **Optionnel** : choisis un **Template** (CAF, Impôts, SFR, Banque...)
5. Ajoute ou retire des champs dans la zone centrale
6. Clique sur **GÉNÉRER MAINTENANT** (bouton orange)

Le dossier `donnees_generees` s'ouvre automatiquement avec ton fichier !

**Astuce** : tu peux réordonner les colonnes avec les boutons ↑ ↓
